## How to Upgrade or Downgrade

**Upgrade:**
1. Dashboard > Account > Plans
2. Select the target plan
3. Enter payment details (if not already on file)
4. Plan activates immediately; prorated charges apply for the current billing cycle

**Downgrade:**
1. Dashboard > Account > Plans
2. Select a lower plan
3. Downgrade takes effect at the end of the current billing cycle
4. If current usage exceeds the lower plan's limits, a warning is shown
5. Files are NOT deleted, but sync pauses if storage exceeds the new quota

## Billing FAQ

**Q: When am I charged?**
A: Monthly plans are charged on the same date each month. Annual plans are charged upfront.

**Q: What payment methods are accepted?**
A: Visa, Mastercard, American Express, and wire transfer (Enterprise only).

**Q: Can I get a refund?**
A: Monthly plans can be cancelled anytime; no refunds for partial months. Annual plans have a 30-day refund window.

**Q: What happens if my payment fails?**
A: You have a 7-day grace period. After that, the account is downgraded to the Free plan. See error code ACCT-001.

**Q: How does per-user pricing work?**
A: You are charged for each active user in your workspace. Deactivated users are not charged. Changes are prorated.
